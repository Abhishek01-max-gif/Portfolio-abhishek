Latest Project

Random Games

The 3 Chance Win Game is a simple two-player game where each player is given three attempts to score randomly between 0 and 10. The player with the higher total score after their three attempts wins the game. The game is fair and ensures that each player has an equal number of attempts. The game is designed to be easy to play, requiring minimal input, with an interactive start and a clear display of each player's score.

PDF Uploder

The PDF Upload application allows users to upload PDF files through a simple interface. The uploaded PDF file can be processed by the backend server (though in this example, we'll focus on the frontend aspect). This project uses HTML, CSS, JavaScript, and Bootstrap to create an interactive, userfriendly web page that lets users select and upload PDF files.

QR Generator

The QR Code Generator is a simple and efficient web-based application that generates QR codes from user-provided text.
